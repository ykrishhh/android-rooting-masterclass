## Features

| Layer | Tools | Use Case |
|-------|-------|----------|
| User-Space | APKTool, Jadx | DEX editing, decompilation |
| Runtime | Frida, Objection | Dynamic instrumentation |
| Framework | Xposed, LSPosed | System-level hooks |
| Kernel | KernelPatch, eBPF | Kernel-level patching |
